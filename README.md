# Favran Cargo Support

Ин папка барои сохтани саҳифаи расмии дастгирии Favran Cargo дар GitHub Pages омода шудааст.

## Чӣ тавр ҷойгир кардан

1. Дар GitHub repository-и нав бо номи `favrancargo-support` созед.
2. Файлҳои `index.html` ва `logo.png`-ро ба repository бор кунед.
3. Ба **Settings → Pages** дароед.
4. Дар **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
5. **Save**-ро пахш кунед.
6. Баъди 1–5 дақиқа саҳифа дар чунин URL дастрас мешавад:

   `https://hia-03.github.io/favrancargo-support/`
